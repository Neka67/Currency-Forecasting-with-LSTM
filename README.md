# 📈 Currency Forecasting with LSTM
A Deep Learning project that predicts USD/TRY exchange rates using Long Short-Term Memory (LSTM) networks. Developed as a personal portfolio project to bridge Economics and Computer Science.

## 📊 Performance
- **RMSE:** 0.34 TL
- **Model:** 2-Layer LSTM with Dropout
- **Data Source:** Yahoo Finance (2015-Present)

## 🚀 Future Vision: "The Seoul Exchanger"
This project is the core engine for an upcoming web application designed for exchange students in South Korea to manage USD/KRW/TRY conversions and gold/silver investments.
